Water Supply Network Monitoring System

02 Completed

ZeroG

The Seamless Gesture Interface

A computer vision system that replaces physical mouse and keyboard input with hand gestures, tracked in real time via webcam. Uses Mediapipe's hand-landmark model to translate finger positions into system-level mouse and keyboard events.

  • 21-point hand landmark tracking using Mediapipe
  • Gesture-to-action mapping: pinch to click, swipe to scroll, spread to zoom
  • Custom gesture vocabulary — user-definable shortcuts
  • Zero hardware dependency — works with any standard webcam
  • Low-latency pipeline optimised with frame skipping

HealthAssist

Your Personal Health Buddy — Android App

An Android application that diagnoses the likelihood of diabetes, anemia, and hypertension from user-reported symptoms using a fuzzy logic inference engine. Designed to assist in regions with limited access to immediate medical consultation.

  • Fuzzy logic reasoning engine for three disease diagnoses
  • Symptom input via conversational question flow
  • Likelihood scoring with confidence levels, not binary yes/no
  • Offline-first — no internet required for diagnosis
  • Clean Material Design UI built in Java for Android
